Credit Card Fraudulent Transaction Detection

Conducted thorough data pre-processing to extract the essential features from a historical transaction dataset, utilizing a correlation matrix. For an in-depth exploration of fraud patterns, I performed Exploratory Data Analysis (EDA) on the initial fraud values across different features. To enhance visual representation, I integrated a PowerBIClient dashboard. Employing statistical analysis, I identified and removed outliers, optimizing data quality. Through the application of supervised learning techniques, I achieved a model with high accuracy by fine-tuning a Random Forest Classifier. The resulting predictions of original fraud values demonstrated an impressive accuracy rate of 89% and an F1-score of 91%, showcasing the effectiveness of the refined model

Database Design and Development for Online Dental Office Service

I orchestrated the development of a comprehensive data model tailored for a dental clinic, encompassing 33 entities along with 27 business rules that delineated both in-scope and out-of-scope parameters. Transforming the conceptual structure into a tangible format, I created an Entity-Relationship (ER) Diagram and subsequently converted it into a script for a SQL Server Management Studio (SSMS) data file. To facilitate seamless interaction with the database, I utilized WampServer to establish connections between web forms and MySQL, incorporating essential Create, Read, Update, and Delete (CRUD) functionalities on the clinic's website
